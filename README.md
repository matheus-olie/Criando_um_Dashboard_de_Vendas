# Criando_um_Dashboard_de_Vendas
Desafio realizado no Bootcamp Santander, pela DIO. A atividade requeria a criação de um Dashboard de vendas em Excel!
---------------------------------------------------------------------------------------------------------------
O desafio constou de:
* Metodo organizacional
* Leitura e entendimento de uma base de dados
* Transformação de dados em informações que respondessem quatro perguntas de negócios:
  I) Qual o faturamento total de vendas de planos anuais (contendo todas as assinaturas agregadas)
  II) Qual o faturamento total de vendas de planos anuais, separados por auto renovação
  III) Total de vendas de assinatura EA play
  IV) Total de vendas de assinatura Minecraft Season Pass
* Gráficos dinâmicos e Filtros visuais
* Organização e Estilização do Dashboard
* Big Numbers
* Info context
